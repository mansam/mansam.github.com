`hotdate`: intuitive date formatting

hotdate is a library I wrote for the first 5 week release cycle of Remy DeCausemaker's Advanced Projects in FOSS class. It's an extension of the builtin Python datetime object that adds a bunch of date formatting and relative date handling in an attempt to make certain common tasks easier to do. Its API is very much inspired by that of moment.js.

It's dependent on six for 2/3 compatibility, and the awesome freezegun for testing. I originally had hoped to avoid other dependencies, but I ended up capitulating and making hotdate dependent on python-dateutil's great support for working out relative time deltas.

Check it out (along with its documentation) on the hotdate Github page.
